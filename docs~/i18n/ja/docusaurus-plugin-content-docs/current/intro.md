U-Stella FaceTracking は、VRChatアバター向けのフェイシャルトラッキング設定を補助するUnityエディター拡張です。
使用するトラッキング機器、使いたい表情機能、顔メッシュのブレンドシェイプ割り当てをInspector上で設定できます。

## 必要環境

- Unity 2022.3
- VRChatアバター用のUnityプロジェクト
- Modular Avatar
- フェイシャルトラッキング用のブレンドシェイプを持つ顔メッシュ
